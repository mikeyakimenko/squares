# Just a beggining

Simple squares app, with love and for fun ;)

## how to run

You need install mongodb also nodejs. Open a three tabs of you're terminal:

**1st**

`cd /path/to/project/api`
`mongod`

**2nd**

`cd /path/to/project/api`
`yarn instal && yarn dev`

**3rd**

`cd /path/to/project/client`
`yarn instal && yarn dev`
